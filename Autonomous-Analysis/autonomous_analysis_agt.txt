2️⃣ Anomaly Detection Agent

This agent monitors KPI behavior.

Methods:

• Z-score detection
• Isolation Forest
• seasonal trend detection

Example:

class AnomalyAgent:

    def detect(self, metric_history):

        mean = metric_history.mean()
        std = metric_history.std()

        latest = metric_history.iloc[-1]

        z_score = (latest - mean) / std

        if abs(z_score) > 3:
            return True

        return False

Agent Orchestrator

All agents run through a central orchestrator.

Example flow:

class KPIAgentOrchestrator:

    def run_pipeline(self, df):

        kpis = DataAnalysisAgent().run(df)

        anomaly = AnomalyAgent().detect(kpis)

        if anomaly:

            root_cause = InvestigationAgent().investigate(df)

            insight = ExecutiveInsightAgent().explain(
                kpis, root_cause
            )

            AlertAgent().send(insight)

            ReportAgent().build(insight)
